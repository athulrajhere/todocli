# todocli

A fast, minimal todo manager for the terminal built with Go.

## Install
```bash
go install github.com/athulrajhere/todocli@latest
```

## Usage
```bash
# Add a todo
todo add "Buy groceries"

# List all todos
todo list

# Mark as complete (interactive)
todo complete

# Delete a todo (interactive)
todo delete
```
